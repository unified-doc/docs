### Overview

<SourceCode source="examples/file-system" />

An interpretation of a simple file system built on the web using `unified-doc`.  This example uses the `DocPreview` component that is detailed in the <Link to="/examples/doc-preview">Doc preview example</Link>.

This example is used in the home page!

### Features
- Features from the <Link to="/examples/doc-preview">Doc preview</Link> example apply here.
- Upload and preview any file using the `DocPreview` component.
- Search/filter for files.
- Hide/show file extensions.

### Concepts
- Most concepts from the <Link to="/examples/doc-preview">Doc preview</Link> example apply here.
- `dom.fromFile()`
