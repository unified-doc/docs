A simple component that renders any document and supports most useful features encountered in document applications.  This example applies all core `unified-doc` and `unified-doc-dom` API methods, and is used throughout the documentation site to render documents.

### Features
- Render any document using `unified-doc`.
- Rendered document is enhanced by relevant plugins and web technologies (syntax highlighting and custom CSS).
- Search and jump to matched results with anchor links and apply a brief highlight effect.
- Preview document in various ways (compiled, text content, syntax tree).
- Download document in various supported formats (source, `.html`, `.txt`).
- Create, manage, and navigate to highlightable bookmarks that persist when `.html` version of the file is downloaded.

### Concepts
- Apply custom `mark` CSS.
- `mark` anchorlinks.
- `doc.compile()`
- `doc.file()`
- `doc.parse()`
- `doc.search()`
- `doc.textContent()`
- `dom.highlight()`
- `dom.registerMarks()`
- `dom.selectText()`
- `dom.saveFile()`
- `options.marks`
- `options.prePlugins`
- `options.sanitizeSchema`
