### Overview

This example shows how we can render `.docx` files by converting them to their HTML representation via the [`mammoth`][mammoth] library.  With the document represented in HTML, we can benefit from all `unified-doc` APIs (e.g. render, search, export, marks etc).

<SourceCode source="examples/docx" />

### Features
- Features from the <Link to="/examples/doc-preview">Doc preview</Link> example apply here.
- Upload and preview any `.docx` file using the `DocPreview` component.

### Concepts
- Most concepts from the <Link to="/examples/doc-preview">Doc preview</Link> example apply here.
- Converting source content to HTML for access to `unified-doc` APIs.
