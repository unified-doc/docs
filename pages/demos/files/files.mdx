# Files
This demo showcases how `unified-doc` can easily render a variety of content previews, and export files in supported formats.  Click on a file icon to view the details of the file.

## Concepts
- Using the `doc.file()` API method.
- Using the `doc.textContent()` API method.
- Rendering supported content types.
- Exporting different file formats.
