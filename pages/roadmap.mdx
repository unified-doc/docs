# Roadmap

Upcoming project development:
- Get feedback and official launch packages out of beta.
- Work on `unified-doc-cli` package to support transforming and working with documents in Node/CLI.
- Work on `unified-doc-parse-pdf` (or maybe `repdf`) package.
- Work on `unified-doc-parse-docx` (or maybe `redocx`) package.
- Discuss and spec `.uni` file format (`text/uni`) with `unified` team.
