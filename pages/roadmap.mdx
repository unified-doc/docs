# Roadmap

Upcoming development goals for the project:
- Formalize `unified-doc-cli` package.
- Formalize `unified-doc-dom` package.
- Discuss and spec `.uni` file format (mime type: `text/uni`) with `unified` team.
- Work on `repdf` (or `unified-doc-parse-pdf`) and `redocx` (or `unified-doc-parse-dox`) packages to parse `.pdf` and `.docx` content respectively to make the ecosystem more accessible to these popular content types.
